```md
# Spacy NER: Product & Quantity Extractor

A custom spaCy NER model to extract product names and quantities from text.



### Example Input
```
"I need 5 bags of rice."
```

### Output
```
[('5 bags', 'QUANTITY'), ('rice', 'PRODUCT')]
